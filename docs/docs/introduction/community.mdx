Instant OpenHIE is an open-source project. We encourage contributions and suggestions! If you would like to get involved, please visit us on [Github](https://github.com/openhie/instant). Feel free to [submit an issue](https://github.com/openhie/instant/issues/new) or to [create a PR](https://github.com/openhie/instant/compare) to see your features included in the project.

Instant OpenHIE is extensible by design. Consider [creating a package](../how-to/creating-packages) to support your particular use case.

We look forward to growing the set of capabilities within Instant OpenHIE together!
