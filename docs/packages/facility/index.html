Facility-Registry

info

The Instant OpenHIE architecture, codebase, and documentation are under active development and are subject to change. While we encourage adoption and extension of the Instant OpenHIE framework, we do not consider this ready for production use at this stage.

caution

This package is still under development

#Package functionality

This package sets up the Global Open Facility Registry - GOFR which acts as a facility registry. Other solutions may be added in future to function as facility registries.

GOFR provides the ability to manage an authoritative facility list and reconcile lists of other systems. Please see the documentation for more about facility registries.
