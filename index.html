Introduction

The Instant OpenHIE project aims to reduce the costs and skills required for software developers to deploy an OpenHIE architecture for quicker initial solution testing and as a starting point for faster production implementation and customization.

View the Introduction section to learn more.

Concepts

Instant OpenHIE provides an easy way to setup, explore and develop with the OpenHIE Architecture. It allows packages to be added that support multiple different use cases and workflows specified by OpenHIE. Each package contains scripts to stand up and configure applications that support these various workflows.

View the Concepts section to learn more.

Packages

The fundamental concept of Instant OpenHIE is that it can be extended to support additional use cases and workflows. This is achieved through packages. A number of essential package are bundled with Instant OpenHIE.

View the Packages section to learn more.
